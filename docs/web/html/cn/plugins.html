相关插件

Micro架构中所有工具都可插拔，也即是说内在实现都是可以剥离出来。

Go-Micro和Micro工具集的插件是各自独立的，可以在导航栏里查看更多的相关信息。

几个例子

Go Micro

  • Etcd Registry - 基于Etcd服务注册发现插件
  • K8s Registry - 基于k8s的服务注册发现插件
  • Kafka Broker - Kafka消息代理（broker）

Micro Toolkit

  • Router - 可配置的http路由、代理
  • AWS X-Ray - 基于AWS X-Ray集成的链路追踪
  • IP Whitelite - IP白名单插件

仓库

相关开源的插件可在github中找到github.com/micro/go-plugins。
